# MapReduce Program

GitHub 데이터를 렌더링 하는 API 서버

## Server Architecture

![Server2](https://github.com/JayStevency/JayStevency/blob/master/PortfolioProject1/Server2.png)


### 서버 주요 기능 :
- Spring-boot
- Restful
- MapReduce

## How to run Server?

### 요구사항 :

- JAVA : 1.8 
- Maven : 3.3.9
- Spring-boot : 1.5.2

### 실행 방법

1. 빌드 방법
    
 > **pom.xml** 파일이 있는 경로에서 ```mvn clean package```

 
2. 빌드 후 run


 >```java -jar target/mapreduce-0.0.1-SNAPSHOT.jar``` 
 >
 >  혹은
 >
 >```mvn spring-boot:run```
